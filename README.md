# Awesome Adventure Library (Local Prototype)

このプロジェクトは現在、データベースを一切使わずにローカル環境のみで動作するプロトタイプ（完全なUIとアップロードファイルのローカル保存機構）として構築されています。

## 次にサーバー構築を担当するAIエージェントへの指示
このリポジトリのフロントエンドUI（本棚の一覧、PDFビューワー機構、画像のプレビューや背表紙のドミナントカラー自動抽出など）は、すでにすべて完成して動作しています。

現在のバックエンド処理（本の「登録」と「取得」処理）は、仮実装として **`src/app/actions.ts`** にまとめられています。
このファイルは現在 `fs/promises` を使ってローカルの `data/books.json` を読み書きし、アップロードされたファイルを `public/uploads/` に保存する仕組みで仮稼働しています。

**あなたのタスクについて:**
フロントエンドのページ (`page.tsx` や `admin/books/page.tsx` など) のUIコードは一切変更する必要はありません。
バックエンド・サーバーインフラの整備やデータベースの導入を行う際は、`src/app/actions.ts` の中身の処理（DBへのクエリなど）を書き換えるだけで、すべてのUIがこれまで通り自然に機能するよう結合部を整理しています。

## 起動方法
```bash
npm install
npm run dev
```

フロントエンドは完全に独立して動いているため、すぐにテスト・確認が可能です。
